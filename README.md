# 🎮 Office Game Hub

Office Game Hub is an offline mini-game collection built entirely with **Python** and **Tkinter**. It includes **10 playable games** in one application, along with office-themed Terminal, Spreadsheet, and Code Editor screens.

No external Python packages are required.

## 🕹️ Games Included

1. 2048
2. Snake
3. Blackjack
4. Tic-Tac-Toe
5. Pong
6. Memory Match
7. Minesweeper
8. Typing Speed
9. Rock Paper Scissors
10. Number Guessing

## ✨ Features

* 10 games in one application
* Built using Python and Tkinter
* Works completely offline
* No external dependencies
* Simple game-selection menu
* Terminal-style interface
* Excel-style spreadsheet interface
* Code Editor-style interface
* Keyboard shortcuts for quick navigation
* Lightweight and easy to run

## 📥 Installation

### 1. Clone the repository

```bash
git clone https://github.com/sidharth1606/Office-Game-Hub.git
```

### 2. Open the project folder

```bash
cd Office-Game-Hub
```

### 3. Run the application

```bash
python office_game_10_games.py
```

On some Windows computers, use:

```bash
py office_game_10_games.py
```

## ▶️ How to Use

After running the Python file, a window named **`Report_Analysis.xlsx`** will open.

The application initially displays a simulated office screen that may look like a Terminal, Excel spreadsheet, or Code Editor. The screen may also display simulated office activity.

Press the **`F2` key** to open the Game Menu.

Select any game from the menu and start playing.

To quickly return to the previously selected office-style screen while playing a game, press the **`ESC` key**.

> **Note:** The `Report_Analysis.xlsx` window is only an office-themed interface created inside the Python application. It does not open, create, read, or modify a real Excel file.

## ⌨️ Keyboard Controls

| Key        | Action                                                |
| ---------- | ----------------------------------------------------- |
| `F2`       | Open the Game Menu                                    |
| `F3`       | Show the Terminal screen                              |
| `F4`       | Show the Excel-style Spreadsheet screen               |
| `F5`       | Show the Code Editor screen                           |
| `ESC`      | Return to the previously selected office-style screen |
| Arrow Keys | Control 2048, Snake, and Pong                         |
| `R`        | Restart Snake after game over                         |

## 🪟 Run Without a Console Window

Windows users can run the application without opening a Command Prompt window:

```bash
pythonw office_game_10_games.py
```
